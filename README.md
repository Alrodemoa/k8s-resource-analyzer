# K8s Resource Analyzer

Инструмент для анализа ресурсов Kubernetes-кластера. Собирает метрики CPU и памяти по узлам, неймспейсам и подам, формирует рекомендации по оптимизации и экспортирует в Excel-отчёт.

## Возможности

- Не требует `kubectl` — использует нативный Go-клиент `client-go`
- Метрики в реальном времени через Metrics Server API
- Excel-отчёт с несколькими листами, цветовой индикацией и рекомендациями
- Интерактивный выбор неймспейсов перед анализом
- Windows, Linux, macOS (включая Apple Silicon)

## Требования

- Настроенный `kubeconfig` (`~/.kube/config` или переменная `KUBECONFIG`)
- [Metrics Server](https://github.com/kubernetes-sigs/metrics-server) в кластере

## Использование

Скачайте бинарник для вашей платформы со страницы [Releases](../../releases).

```bash
# Запуск с буфером 50% (по умолчанию)
./k8s-analyzer

# Задать буфер вручную (% запаса при расчёте рекомендаций)
./k8s-analyzer -b 30    # dev-кластеры
./k8s-analyzer -b 100   # prod-кластеры

# Справка
./k8s-analyzer --help
```

При запуске инструмент предложит выбрать неймспейсы для анализа:

```
Введите номера (1,3 / 1-4 / *):
```

Результат — файл `k8s-анализ-YYYY-MM-DD-HHmmss.xlsx`.

## Структура отчёта

| Лист | Содержимое |
|------|------------|
| Сводка кластера | Общая статистика, CPU/память |
| Узлы | Ёмкость, нагрузка, распределение запросов |
| PVC | Persistent Volume Claims: статус, ёмкость |
| PV | Физические тома: привязки, статус |
| `<namespace>` | Детализация по подам с рекомендациями |

Статусы эффективности ресурсов:

| Статус | Условие |
|--------|---------|
| Критично | ≥ 100% от запроса |
| Высокое | ≥ 80% |
| Норма | ≥ 50% |
| Низкое | ≥ 30% |
| Минимум | < 30% — сильно переоценён |

## Сборка из исходников

Требуется Go 1.19+.

### Через Makefile (Linux / macOS)

```bash
make install      # загрузить зависимости
make build        # собрать для текущей платформы → bin/k8s-resource-analyzer
make build-all    # собрать для всех платформ → bin/
make run          # запустить без сборки (go run)
make clean        # удалить bin/
```

### Вручную

**Linux / macOS:**
```bash
go mod download
go build -o k8s-analyzer ./cmd/k8s-analyzer
```

**Windows:**
```cmd
go mod download
go build -o k8s-analyzer.exe .\cmd\k8s-analyzer
```
